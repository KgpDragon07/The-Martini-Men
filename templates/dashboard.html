redDash

Interactive help dashboard to provide you an overview of the ongoing disaster situation
When we are struck by any natural disaster our first impulse is to provide first aid and help of any kind as soon as possible. But, today, with the huge growth in Machine Learning industry and a rich Open Source community we can do a lot more to predict, manage and assess disasters.
Maps: We put the NASA open source toolkit and dynamically logged csv files, which are updated daily with disaster affected location data, to use by cleaning them and displaying high alert zones on a Map.
As of now it takes flood data and generates the map accordingly.
Upload: Check whether a snapshot or an image from social media or a drone is suggestive of an ongoing natural disaster. We use Microsoft Azure Custom Vision API to classify images from different sources.
Chat: We provide a simple Android app, redType, with basic SMS functionality. During time of distress, it would help send SOS messages to our Helpline number along with the Location of the sender.
The Helpline admin can the reply through our interface back to the victim, stating the course of the rescue operation.
